
# Verbum Environment

### <b>Requirements</b>

- Electron v22.1.0 (Stable) [Download](https://github.com/electron/electron/releases/tag/v22.1.0)
- git, wget

<br>

### <b>Install</b>

Tested on Linux Mint x64.

```bash
git clone https://github.com/melchisedech333/verbum-environment.git
cd verbum-environment
chmod +x install.sh
./install.sh
```
